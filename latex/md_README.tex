\end{DoxyItemize}\hypertarget{md_README_autotoc_md1}{}\doxysection{Features}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item mcpp (Math for C Plus Plus) is {\bfseries{cross platform}}, {\bfseries{templated math library}} that can be used without compiling .
\item It is targeted towards {\bfseries{engineers}} and {\bfseries{mathematicians}}.
\item Used with a very simple and intuitive syntax inspired from Python libraries such as \href{https://www.sympy.org/en/index.html}{\texttt{ sympy}}, \href{https://numpy.org/}{\texttt{ numpy}}, \href{https://www.scipy.org/}{\texttt{ scipy}} .. etc
\item By only including {\ttfamily mcpp.\+hpp} file you get all the functions with their implementations into your code
\item Unit testing is done using \href{https://github.com/google/googletest}{\texttt{ Google test}}
\item Main targeted math sectors \+:
\begin{DoxyItemize}
\item Algebra
\item Statistics
\item Calculus
\item Vectors
\item Signal systems
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{Compiling}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item There is {\bfseries{NO NEED FOR COMPILING}} just include the header files right away.
\item You can use the compiling procedure to install everything to your standard path
\item The compiling procedure is only used for the test scripts
\item Tested Platforms
\begin{DoxyItemize}
\item Linux Ubuntu 18.\+04
\item mac\+OS 10.\+13 with xcode 9.\+4.\+1
\item Windows 10 version 1809
\end{DoxyItemize}
\item Dependencies
\begin{DoxyItemize}
\item cmake version 3.\+13 or higher
\item GNU c++ compiler version 6.\+1 ot higher (C++ 14 features)
\item GNU make version 4.\+1 or higher
\end{DoxyItemize}
